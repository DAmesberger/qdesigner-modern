# Appendix A: Keyboard Shortcuts

Complete reference of keyboard shortcuts available in the QDesigner Modern designer interface.

---

## Command Palette

Open the command palette with `Ctrl+K` (or `Cmd+K` on macOS) to access all commands by name. The palette supports fuzzy search -- type any part of a command name to filter the list.

---

## Add Questions

| Shortcut | Action |
|---|---|
| `T` | Add Text Input Question |
| `M` | Add Multiple Choice Question |
| `R` | Add Reaction Time Question |
| `P` | Add Page |

These single-key shortcuts are available when the command palette is open or when no text input field has focus.

---

## Edit

| Shortcut | Action |
|---|---|
| `Ctrl+Z` | Undo |
| `Ctrl+Shift+Z` | Redo |
| `Ctrl+D` | Duplicate Selected Item |
| `Del` | Delete Selected Item |

---

## View

| Shortcut | Action |
|---|---|
| `Ctrl+=` | Zoom In |
| `Ctrl+-` | Zoom Out |
| `Ctrl+0` | Reset Zoom |

The designer supports two view modes accessible via the command palette:

- **Visual View** (WYSIWYG): Shows the questionnaire as participants will see it.
- **Structure View**: Shows the hierarchical structure of pages, blocks, and questions.

---

## Actions

| Shortcut | Action |
|---|---|
| `Ctrl+S` | Save Questionnaire |
| `Ctrl+Shift+Enter` | Publish Questionnaire |
| `Ctrl+P` | Toggle Preview |

---

## Panel Navigation

The following commands open specific panels in the designer sidebar. They are accessible through the command palette:

| Command | Panel |
|---|---|
| Open Structure Panel | Shows the page/question hierarchy |
| Open Add Panel | Shows the question type palette |
| Open Variables Panel | Shows variable definitions and formulas |
| Open Flow Panel | Shows flow control and skip logic |

---

## Command Palette Navigation

| Key | Action |
|---|---|
| `Arrow Down` | Select next command |
| `Arrow Up` | Select previous command |
| `Enter` | Execute selected command |
| `Escape` | Close command palette |
